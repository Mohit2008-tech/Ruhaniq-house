Build a beautiful, heart-touching single-page website (Phase 1) for a handmade products brand called **Ruhaniq** using React + Tailwind CSS.

---

BRAND IDENTITY
- Name: Ruhaniq (meaning: soulful / spiritual in Urdu)
- Tagline options (use all, rotate or stack):
  • "Pretty things with a soul"
  • "Nazakat in every detail" (Nazakat = elegance/delicacy in Urdu)
  • "Handcrafted with love and warmth"
- Aesthetic: Warm, romantic, artisanal. Think soft creams, dusty roses, warm beiges, terracotta, and sage green. The visual mood should feel like a beautifully lit handmade studio — warm light, soft textures, heartfelt and premium.
- Fonts: Use Google Fonts — "Cormorant Garamond" (serif, for headings/taglines) + "Lato" or "DM Sans" (sans-serif, for body text). This pairing feels elegant and artisanal.

DESIGN RULES
- Color palette: cream (#FDF6EE), blush pink (#F2C4CE), dusty rose (#C97B84), warm beige (#E8D5B7), terracotta (#C4704F), sage green (#9CAF88), warm white (#FFFAF5)
- NO cold blues, no neon colors, no stark blacks — keep everything warm and soft
- All corners rounded (use rounded-2xl or rounded-3xl generously)
- Generous padding and whitespace — the page should breathe
- Hover effects: soft scale + warm shadow on cards/buttons
- All images are placeholder slots — use a warm-toned placeholder gradient with a centered label like "[Upload product photo]" using a dashed border in dusty rose
- Fully responsive — mobile first
- Smooth scroll behavior
- Subtle entrance animations (fade-in on scroll using Intersection Observer or AOS library)

---

PHASE 1 GOALS
This is Phase 1 — the goal is to present the brand beautifully and build emotional connection. Do NOT add a shopping cart, login, or backend. Just a stunning, heartfelt landing page where the owner can later swap in real photos, videos, and a price menu.

Make it look like a premium handmade brand from Day 1.
